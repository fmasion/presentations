...Bières

Scala-js c'est quoi ?

C'est un compilateur transformant du code scala en javascript

c'est tout !

Comment çà marche ?

c'est un plugin du compilateur scala

  • Remplace la dernière étape du compilateur
  • => Génère du js à la place du bytecode
  • Utilise la suite de test du compilateur scala
  • Scala-js a rejoint les projets dbuild

Conséquences

Tout scala est supporté

  • les classes, objects, traits
  • les imports
  • les implicits
  • les macros !!!
  • ...

Warning

Limitations 1/3

Liées à l'environement d'execution

http://www.scala-js.org/doc/semantics.html

  • Nécéssite ES5.1+
  • Pas de Thread
  • Exceptions supp. en fullOpt
  • Mapping de Types Primitifs vers les types JS
  • Pas de Réflexion
  • Regexp
  • Pas de Weak Reference

Limitations 2/3

Liées au compilateur

  • Compile à partir du source ou archives SJS
  • Ne compile pas les sources Java
  • Ne converti pas les Jars
  • Format "binaire" specifique
  • Pas de compatibilité binaire entre version de scala ET de scala-js
  • Taille du js

Limitations 3/3

Liées à l'écosysteme

  • Jeunesse => couche low level
  • Pas encore de framework web idiomatique
  • Des wrappeurs encore buggés
  • Tous les Type de base de la JVM ne sont pas convertis
  • Les librairies compatibles ne publishent pas encore pour SJS...
  • Setup sbt...
